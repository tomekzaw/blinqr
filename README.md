# blinqr
A prototype implementation of an unidirectional connectionless stateless data transfer protocol from a computer to a mobile device.

## Requirements
* Python 3.7
* zbar 0.10
* Tkinter
* DroidCam 6.0

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python sender.py
```
```bash
python receiver.py
```
